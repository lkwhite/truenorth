Understanding tRNA Nomenclature

          Example: tRNA-Ala-AGC-12-3

                        |   |   |  |

                        |   |   |  +-- Copy number (3rd copy)

                        |   |   +----- Gene family (family 12)

                        |   +--------- Anticodon (AGC)

                        +------------- Amino acid (Alanine)
ISOACCEPTOR
All tRNAs with the same anticodon (e.g., all Ala-AGC tRNAs). These decode the same codon(s).
ISODECODER
tRNAs with the same anticodon but different body sequences. Important for probe specificity! Two isodecoders may not cross-hybridize to the same probe.
GENE FAMILY
Copies from the same ancestral gene (e.g., Ala-AGC-1-1, Ala-AGC-1-2, Ala-AGC-1-3 are all family 1). Usually have identical or nearly identical sequences.
Probe Design Tip: To hit an entire gene family, design a probe against any member. To discriminate between isodecoders, look for regions where their sequences differ.
